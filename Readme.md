# WebDeskLinker

WebDeskLinker는 웹 기반 바탕화면을 모방하여 사용자가 손쉽게 다양한 링크에 접근할 수 있도록 도와주는 프로젝트입니다.

## 주요 기능

- 바탕화면처럼 정리된 인터페이스에서 웹사이트로의 빠른 접근 가능
- 모달 형태로 블로그를 바로 확인할 수 있는 기능
- 간편한 링크 관리 및 추가 기능

## 사용 방법

1. 해당 프로젝트를 클론합니다.

```bash
git clone https://github.com/alsgudrl132/WebDeskLinker.git
```

2. 해당 프로젝트 경로로 이동합니다.
   cd blog

3. **프로젝트 빌드 및 실행:**
```bash
./gradlew build
./gradlew bootRun
```
4. 열린 웹사이트로 이동
   localhost:8080
